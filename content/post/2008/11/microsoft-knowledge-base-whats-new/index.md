So often you find yourself with a technical problem and after long searching you might come across a Microsoft Knowledge base article. Especially with new products so called quick fix engineering updates are provided on a regular basis, so I was wondering how do you get to know about such QFE's ?. I finally came across the [KBUpdate.info ](http://kbupdate.info/)website which is scanning the Microsoft Knowledge Base every night.
