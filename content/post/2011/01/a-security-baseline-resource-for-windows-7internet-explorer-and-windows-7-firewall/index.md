If you are looking for some ideas how to secure your Windows 7 clients, have a look at the [USGCB](http://usgcb.nist.gov/) (The United States Government Configuration Baseline). The USGCB has been developed by the Department of Defense (DoD) and the National Institute of Standards and Technology. The documentation looks impressive and even if you aren’t going to apply all of these 1-1, it might give you some ideas on how to make your clients more secure. 

                       Some of you might remember the security guides [provided](http://www.nsa.gov/ia/guidance/security_configuration_guides/operating_systems.shtml#microsoft) by the NSA (National Security Agency) for Windows NT, 2000, XP and Vista, but you won’t find a Windows 7 Security guide there, this because The "Specialized Security - Limited Functionality" (SSLF) security settings in Microsoft's *Windows Vista/7 [Security Guide](http://technet.microsoft.com/en-us/library/ee712767.aspx)* track closely with the security level historically represented in the NSA guidelines.
