if you ever need to analyze a Windows Dump file but don’t have the Windows Debugging Tools available here’s a handy way of doing an online analysis of the dump file.

  ![image](images/image_thumb2.png)

  Get the Windows Crash Dump file from C:\Windows\Minidump, then ZIP the dump file and upload it on the OSR Online website via the Instant Online Crash Analysis Service.

  [http://www.osronline.com/page.cfm?name=analyze](http://www.osronline.com/page.cfm?name=analyze)

  ![2011-03-27 15h14_22](images/2011-03-27-15h14_22_thumb.png)

  After a while you get an online analysis report.

  ![2011-03-27 15h14_48](images/2011-03-27-15h14_48_thumb.png)
