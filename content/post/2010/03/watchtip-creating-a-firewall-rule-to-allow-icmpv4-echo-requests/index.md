Windows 7 with the Firewall enabled by default does not allow ping requests, hence when you ping a remote Windows 7 client it will not respond. 

  In this WebCast [John Baker](http://blogs.technet.com/johnbaker/) explains how to configure the Firewall to respond to ICMPv4 Echo Requests. 

  Watch the WebCast [here](http://edge.technet.com/Media/Creating-a-Firewall-Rule-to-Allow-ICMPv4-Echo-Requests/)
