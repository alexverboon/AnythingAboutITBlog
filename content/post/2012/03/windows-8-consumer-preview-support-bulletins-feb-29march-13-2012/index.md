Below are the Windows 8 Consumer Preview support bulletins I was able to capture since the release of the Consumer Preview build. 

  The BITS service crashes during the installation of Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2680336](http://support.microsoft.com/kb/2680336)

  Windows 8 Consumer Preview language packs are available for computers that are running Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2607607](http://support.microsoft.com/kb/2607607)

  Windows Internet Explorer 10 Consumer Preview is now available for Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2650043](http://support.microsoft.com/kb/2650043)

  An update for PlayReady PC Runtime for Windows Media Center on a Windows 8 Consumer Preview-based computer is available   
[http://support.microsoft.com/kb/2658597](http://support.microsoft.com/kb/2658597)

  An update is available for the Broker Infrastructure Service (bisrv.dll) in Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2680358](http://support.microsoft.com/kb/2680358)

  Conflicts may occur when you install the Chinese Traditional IME from Office 2010 or from the 2007 Office system on a computer that is running Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2680376](http://support.microsoft.com/kb/2680376)

  The playback of the premium TV content does not work in Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2680328](http://support.microsoft.com/kb/2680328)

  Some apps may stop working several days after installation in Windows 8 Consumer Preview    
[http://support.microsoft.com/kb/2680826#appliesto](http://support.microsoft.com/kb/2680826#appliesto)

  Windows 8 Consumer Preview may crash when you wake Samsung Series 7 Slate   
[http://support.microsoft.com/kb/2684754](http://support.microsoft.com/kb/2684754)

  An update to improve the performance tracking data for the Windows Store in Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2680330](http://support.microsoft.com/kb/2680330)

  You cannot start an application from the tile in Windows 8 Consumer Preview   
[http://support.microsoft.com/kb/2683474](http://support.microsoft.com/kb/2683474)

  .
